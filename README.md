# facedetect
How to download, build, and execute on Jetson nano

$ git clone https://github.com/2sungryul/facedetect.git

$ cd facedetect

$ make

$ ./facedetect

What to do on Windows

execute the command prompt and download gst1.bat in a working directory

\> gst1.bat
